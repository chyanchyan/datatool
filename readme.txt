环境配置：
python 3.6+
脚本会自动检查是否安装pandas和numpy，若无则自动从清华源下载安装

使用方法：
先在data_sources页填写数据名称与路径信息，数据名称与后面的计算逻辑中的引用有关
然后在logics页填写想要计算的统计量和计算规则
然后在命令行进入工具所在文件夹，运行：
python data_tool.py
计算结果会保存在工具所在文件夹下 results.xlsx里

其中：
路径中"./"代表当前目录，"../"代表上级目录，规则与python逻辑一致
数据列列名需要和表头一致，注意全角半角
简单逻辑中，不支持引用多列数据
简单模式优先于advance模式，advance模式下是调用python eval函数求值
